frontend
=========

## Rep-Install
1. Install Gulp Globally
```sh
sudo npm install -g gulp
```

2. Install jspm Globally
```sh
sudo npm install -g jspm
```

## Project Install
1. Install all dependent node packages
```sh
npm install
```

2. Install jspm packages
```sh
jspm install -y
```

## Running App
1. Start Frontend Watch
```sh
gulp watch
```
