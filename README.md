jstty.com
=========

## Rep-Install
1. Install Gulp Globally
```sh
$ npm install -g gulp
```

2. Install jspm Globally
```sh
npm install -g jspm
```

## Project Install
1. Install all root dependent node packages
```sh
npm install
```

2. Install all Frontend and Server dependent packages
```sh
gulp install.all
```

## Running App
1. Start Frontend Watch and Server Watch
```sh
gulp watch
```
